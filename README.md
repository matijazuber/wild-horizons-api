# Wild Horizons

A simple REST API for exploring travel destinations around the world. Search and filter by continent or country to discover new places.

## API Endpoints

- `GET /api` — Retrieve all destinations
- `GET /api/continent/:continent` — Filter destinations by continent
- `GET /api/country/:country` — Filter destinations by country

## Tech Stack

- Node.js
- Native `http` module

## Getting Started

```bash
git clone https://github.com/your-username/wild-horizons.git
cd wild-horizons
node server.js
```

Server runs on `http://localhost:8000`
